# CRM System – Lead & Customer Management Platform

A full-stack **Customer Relationship Management (CRM)** system built using **Spring Boot, React, and PostgreSQL** for managing leads, contacts, follow-ups, and customer interactions.

# Features

- Lead management
- Customer contact management
- Follow-up tracking
- Activity history
- Dashboard analytics
- Role-based access control

### User Roles

**Admin**
- Manage users
- View all leads
- Analytics dashboard

**User**
- Manage assigned leads
- Track follow-ups
- Update customer interactions
